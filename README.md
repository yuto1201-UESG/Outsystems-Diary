# Outsystems-Diary

##  Outsystemsで日記アプリを作る！

ローコードツールで簡単な日記アプリを作ってみた。
今回はモバイル用のアプリを作成。いくつかプラグインを追加した。

環境
*  Macbook air(M1)
*  MacOS 14.4
*  Outsystems(Version 11.54.51)

使用プラグイン
1.  Camera Plugin
2.  Horizontal Date Picker
3.  Lightbox Image
